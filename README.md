# Customer Churn Prediction

## Project Overview
This machine learning project predicts whether a telecom customer is likely to leave the service.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Visualization
4. Data Preprocessing
5. Model Training
6. Prediction & Evaluation

## Result
The model achieved approximately 80% prediction accuracy.

## Dataset
Telco Customer Churn Dataset
